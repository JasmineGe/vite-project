# 1. 技术选型
- vue3 框架 : 采用 vue 框架最新版本 , 组合式 API 形式开发项目。
- vite: 构建化工具
- TypeScript: TypeScript 技术栈
- vue-router: 采用 vue-router 最新版本管理路由
- pinia: 采用 pinia 集中式管理状态
- element-plus: UI 组件库采用 element-plus
- axios: 网络交互
- echarts: 数据可视化大屏 ....

# 2. 学习的收获
1. 掌握如何规范的开发项目
2. 掌握自己如何封装后台管理系统模板
3. 掌握菜单权限与按钮权限
4. 掌握数据大屏该如何解决
5. 掌握请求响应拦截器该如何二次封装
6. 掌握 svg 矢量图如何在项目中使用
7. 掌握主题颜色切换与暗黑模式的切换 .......
